前几天莉莉丝的《海滨消消乐》上线的时候，我发现这个游戏启动界面中居然没有 Unity Logo。 <!--more-->

胡扯游戏群友们聊着聊着，就有技术大佬给出了去掉 Unity Logo 的方法。

下面的方法我也没有试过，请有条件的同学尝试后给我回复下：

**方法一 by @猫小姐（搬砖ing）**

专业版就可以把那个 Power by Unity 标记去掉。 

看得出猫小姐是个有钱人。

**方法二 by @Bacoo**

game.js 里面：

``` javascript
barConfig.style.width = 1
barConfig.style.height = 1
textConfig.style.fontSize = 0
```

iconImage注释掉，那个最开始的进度条就看不见，首包加载的 Logo 也不显示了。

如果还要去掉启动的 Logo，可以修改小游戏配置：`hideAfterCallmain` 这个选项的勾。

如果还要支持 H5 版本中去掉 Unity Logo，可以直接改 unity_default_resources 包里面的那个 Splash，替换一下。
