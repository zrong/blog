## Kimi Code K2.5

Andante 会员，49每月，官网说明每 5 小时能支持 300-1200 次 API 请求。

**完全不够用。** 单 Agent 下，20分钟就触发了 5 小时限额。不到 2 天就干完了一周的额度。

顺便说一句，Kimi 开源的 Kimi Code CLI / Kimi Code for VS Code 编程工具，我测试无法正常使用（网络RETRY），但通过 Claude Code + API Key 是可以正常使用的。
